## Naming Concept ##

Home Assistant Type | Pattern | Hint
------ | ------ | ------
Default | ```<room>_<type>_<attr>_<name>``` |
Scene | ```<room>_<purpose>``` | scene.
Sensor | ```<room>_<sensortype>``` | sensor.
Input Text | ```<room>_<purpose>``` | Purpose must be only small letters
