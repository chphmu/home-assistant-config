# Home Assistant Config by chphmu #

## Naming Concept ##

Home Assistant Type | Pattern | Hint
------ | ------ | ------
Default | ```<room>_<type>_<name>``` |
Scene | ```<room>_<purpose>``` |
Input Text | ```<room>_<purpose>``` | Purpose must be only small letters
